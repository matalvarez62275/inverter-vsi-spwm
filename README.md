# Three-Phase Inverter (VSI) – Induction Motor Control

Este repositorio contiene la documentación, simulaciones y archivos de diseño de hardware de un inverter trifásico desarrollado como proyecto práctico universitario en electrónica de potencia.

## Descripción del proyecto
El proyecto aborda el diseño, simulación e implementación de un inverter trifásico de tensión (VSI) para el control de velocidad de un motor de inducción trifásico, integrando análisis teórico y validación experimental.

## Características principales
- Topología VSI trifásica
- Estrategias de modulación:
  - Onda cuadrada
  - SPWM con control V/F constante
- Modelado y simulación en MATLAB/Simulink
- Análisis del contenido armónico (incluyendo efectos de la 3ª armónica)
- Implementación de la etapa de potencia con MOSFETs y drivers IR2110
- Bus DC a partir de red rectificada con filtrado bulk y de alta frecuencia
- Diseño de PCB y validación
  
## Herramientas y tecnologías
- MATLAB / Simulink
- Electrónica de potencia
- Inverter VSI trifásico
- SPWM
- MOSFETs discretos
- Drivers IR2110
- Diseño de PCB y pruebas de laboratorio

## Notas
Proyecto desarrollado con fines académicos, enfocado en la validación de conceptos fundamentales de electrónica de potencia y control de motores.
